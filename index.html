Progetti

Wolfenstein Web Game

Clone del primo gioco "pseudo-3D" ispirato a Wolfenstein. Sviluppato in HTML/CSS, JavaScript e PHP con backend SQL, include un editor di livelli, sistema di login e salvataggio punteggi.

Amazon Price Tracker (Bot Telegram)

Bot sviluppato in Python per il monitoraggio dei prezzi su Amazon. Notifica via Telegram quando i prezzi scendono. Include scraping periodico e database SQL.

Amazon Price Tracker (JavaFX App)

Applicazione desktop in Java con interfaccia JavaFX per tracciare i prezzi dei prodotti Amazon. Salvataggio dati in SQL, grafico storico, configurazione XML e logging remoto.

Quiz TCP Client/Server

Applicazione in C con socket TCP. Il server multithreaded gestisce più client per un quiz interattivo a risposta multipla. Comunicazione via testo e segnali binari.

Smart Mobility Database

Progetto SQL per la gestione di car sharing, pooling e ride sharing. Include utenti, veicoli, prenotazioni, tragitti e calcolo dei costi su schema E/R normalizzato.
